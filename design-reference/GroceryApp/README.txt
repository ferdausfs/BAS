Grocery App — plain HTML/CSS/JS (কোনো build step নেই)
======================================================

SPCK Editor এ টেস্ট করা:
1) GroceryApp.zip আনজিপ করো
2) SPCK → Open Folder → GroceryApp ফোল্ডার সিলেক্ট করো
3) index.html ওপেন করে Preview চাপো — অ্যাপ চলবে

ফাইল স্ট্রাকচার:
  index.html        → অ্যাপ শেল (এখানেই সব লোড হয়)
  css/style.css     → পুরো ডিজাইন সিস্টেম (মক থেকে মেপা)
  js/icons.js       → সব SVG আইকন
  js/data.js        → সব কনটেন্ট/ডেটা — নিজের ডেটা এখানে বসাও
  js/screens.js     → সব স্ক্রিন রেন্ডার ফাংশন
  js/app.js         → রাউটার (#/route) + ইন্টারঅ্যাকশন

নিজের ডেটা বসানো:
  js/data.js ফাইলে window.D অবজেক্টে সব টেক্সট/প্রোডাক্ট/অর্ডার আছে।
  এখানে বদলালেই পুরো অ্যাপে রিফ্লেক্ট করবে। ছবির জায়গাগুলো এখন
  ফাঁকা গ্রে বক্স — পরে <img> বসানোর জন্য .pimg/.oimg/.lt/.ptb/.msg-img
  ক্লাসগুলোতে ইমেজ যোগ করলেই হবে।

নোট:
- Poppins ফন্ট Google Fonts CDN থেকে আসে — প্রথম লোডে ইন্টারনেট লাগবে।
  অফলাইনে সিস্টেম ফন্টে ফলব্যাক হবে (লেআউট একই থাকবে)।
- বাকি সবকিছু ১০০% অফলাইন চলে।
